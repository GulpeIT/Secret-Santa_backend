# API Тайный санта

Этот репозиторий содержит проект написанный на `Python` к новому году. Необходим он для поднятия back-end в котором задержится всё необходимое для проведения игры **"Тайный Санта"**.

### Основной стек технологий:
| Библиотека/Инструмент| Версия        |
|----------------------|---------------|
| Python               | `>= 3.13.0`   |
| SQLAlchemy           | `>= 2.11.5`   |
| FastAPI              | `>= 0.115.12` |
| Pydantic             | `>= 2.9.1`    |
| Pydantic-Settings    | `>= 2.9.1`    |
| Alembic              | `>= 1.17.2`   |
